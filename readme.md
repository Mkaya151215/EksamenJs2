# Fotball Manager Backend Tjenester

Denne backend-tjenesten er bygget med Node.js og Express for å håndtere API-forespørsler for Fotball Manager-applikasjonen.

## Krav

Før du starter, sørg for at du har følgende installert:

- [Node.js](https://nodejs.org/en/download/)
- [npm](https://www.npmjs.com/get-npm)

## Installere avhengigheter

Naviger til prosjektkatalogen din og kjør følgende kommando for å installere nødvendige avhengigheter:

```bash
npm install


## Starte serveren
For å starte serveren, kjør følgende kommando:
node index.js

Du skal se meldingen Server is running on port 3000, noe som indikerer at serveren kjører på port 3000.

Tilgang til APIet
Når serveren kjører, kan du få tilgang til APIet på http://localhost:3000


## Kjøring av prosjektet
Åpne index.html-filen for å starte prosjektet.
Registrer deg på siden som åpnes.
Etter registrering kan du opprette laget ditt og se detaljer.
